### requirements

```
nodejs v10.11.0 or higher.
npm 6.4.1 or higher.
```

To use:
```
npm install;
npm start;
```

To test:
```
npm test;
```

To build a release:
```
npm run dist;
```

### helpful to find stray packages

npm ls node-gyp
